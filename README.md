# Holiday Experience Website

A responsive travel and vacation booking website built with HTML, CSS, and Bootstrap 5.  

## Features

- **Landing Page:** Hero section with background image carousel and search bar.
- **Navigation:** Multi-level dropdown menus, search, sign in/up links, and Add Listing button.
- **Sections:**
  - Vacation rentals overview
  - Guides for popular cities with clickable cards
  - Last minute deals carousel
  - Luxurious rooms carousel
  - Testimonials carousel
  - Travel blog highlights
  - Instagram gallery
  - Footer with directory links and payment icons
- **Responsive Design:** Mobile-friendly with breakpoints at 768px and 480px.
- **Dynamic Elements:** 
  - Year auto-update in footer
  - Hover effects on cards and images
  - Bootstrap carousel for rooms and testimonials

## Installation

1. Clone the repository:
git clone <repository-url>
Open index.html in a browser.

Ensure images/, style.css, and script.js are in the same directory.

Usage
------------
Use the search bar to filter rentals by name, location, or category.
Click city cards to open Google Maps.
Navigate through rooms and testimonials using carousel arrows.
Explore the blog and Instagram sections for travel content.
